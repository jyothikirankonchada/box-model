Explanation of CSS Box-Model
1)What is CSS Box-Model?
=>CSS box model is a container which contains multiple properties.

2)What are the uses of CSS Box-Model?
=> It is used to create the design and layout of web pages. It can be used as a toolkit for customizing the layout of different elements.

3)What are the properties in CSS Box-Model?
=>borders,margins,padding,Content..etc.
